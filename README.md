### URL LIST CLEANER

Alat pembersih daftar URL berbasis web yang canggih, cepat, dan elegan. Dirancang khusus untuk mengelola, menyaring, dan merapikan ribuan URL sekaligus dengan fokus pada efisiensi dan estetika ala "Mr.Rm19".

---



### 🛠️ Pembersihan & Ekstraksi Tingkat Lanjut
* **Remove URL Parameters**: Menghapus query string (contoh: `?id=123`) secara instan untuk mendapatkan URL bersih.
* **Remove IP-Based URLs**: Mendeteksi dan menyaring alamat IP server secara otomatis.
* **Extract URLs**: Mengekstrak URL secara cerdas dari tumpukan teks yang berantakan menggunakan Regex.
* **Add https://www**: Menyeragamkan protokol dan subdomain pada seluruh daftar URL.
* **Root Domain & Trim to Root**: Mengambil domain inti atau memotong jalur folder (path) dengan cepat.

### 🛡️ Deduping & Filter
* **Remove Duplicate URLs**: Menghapus baris duplikat standar secara akurat.
* **Remove Duplicate Domain**: Menyisakan hanya satu URL unik untuk setiap domain yang berbeda.
* **Remove https Variations**: Saringan cerdas untuk situs yang sama namun menggunakan protokol berbeda (http vs https).

### 📊 Pengurutan & Organisasi
* **Multi-Criteria Sort**: Urutkan berdasarkan Baris, Domain, TLD, Panjang Karakter, atau Acak (Shuffle).
* **Real-time Counter**: Pantau jumlah URL secara dinamis dengan penghitung baris otomatis.

### 🎨 UI/UX Premium
* **Dark/Light Mode Toggle**: Beralih tema gelap atau terang sesuai kenyamanan mata kamu.
* **Mr.Rm19 Progress Bar**: Animasi visual ala terminal saat memproses data, memberikan kesan profesional.
